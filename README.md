#	What is POMOTODO ?
#### POMOTODO는 일정 관리에 특화된 여러기능을 한데 묶은 웹애플리케이션입니다

#### POMOTODO의 기능
 ###### 1. 구글 검색
 ###### 2. 포모도로 타이머
 ###### 3. 투두 리스트
 ###### 4. 낫투두 리스트
 ###### 5. 과거 기록 시각화
 ###### 6. 집중에 도움이 되는 생활 소음
 ###### 7. 크롬 확장 프로그램

## 개발이유  
현재 나에게 가장 필요한 서비스??
공부를 좀 더 효율적으로 하기 위해 매시간 체크하는 pomodoro타이머,
명확한 목표를 설정하기 위한 TodoList,
나의 안좋은 습관을 체크하고 고치기 위한 Not TodoList,
집중하는데 도움이 되는 생활 소음
작은 보상을 얻기 위한 과거 기록 체크
이것들을 한데 묶어 관리 및 사용할 수 있는 웹애플리케이션을 만들고 싶었음

매일 직접 사용하고 있는 프로그램들이기에 사용자의 시선에서 좀 더 냉정하게 평가할 수 있고,
공부했던 내역이 남는다면 많은 도움이 될 것이라고 판단했음.


## 개발목표  -- 예시작성하자 --
내가 가장 필요하고 잘 사용할 프로그램 만들기
통일성 있는 디자인, 폰트, 컬러, 아이콘을 사용하고 적당한 입체감을 표현한다

  디자인 
  둥근느낌의 뉴모피즘 디자인에 베이스가 되는 컬러를 선택 한 뒤 그에 어울리는 색을 추가해서 완성한다

각 기능들이 시너지 효과를 일으킬 수 있도록 구현한다

사용자의 입장에서 편리하고 당연하도록 개발한다
상호작용(버튼), 버튼위치, 불편할만한 문제 미리 생각해서 해결하기(검색창, 모달창 외부 클릭해서 닫기)

과거의 기록을 통해 지속적인 피드백과 작은 보상을 얻기

다양한 디바이스에서 실행가능하며 다양한 플랫폼을 통해 배포하기
